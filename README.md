## React로 Todo-List 만들기

### 컴포넌트 구조

- src
  - App.jsx (Main File)
  - App.css (Main CSS)
  - components
    - layout
      - Header.jsx
      - MainPage.jsx
    - todo
      - AddTodoForm.jsx
      - ShowTodoList.jsx
      - TodoList.jsx
      - TodoItemButtonController.jsx

### 실행 방법

1. Install node modules : `npm i` or `npm install`

2. Run server : `npm start` or `yarn start`
